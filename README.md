# Supermarket App

## Commands

Prerequisites: 
- [pnpm installed](https://pnpm.io/installation)
- (npm may be used as well, just replace `pnpm` with `npm`)

```bash
cd backend

# install dependencies
pnpm install

# starts server at port 3000
pnpm run start
```

```bash
cd web-app

# install dependencies
pnpm install

# start dev server at port 5173
pnpm run dev
```
